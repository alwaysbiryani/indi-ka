# 🇮🇳 Indi-क — The Home for Hinglish

[Live Demo](https://indi-ka.vercel.app/) • Production Ready • Mobile-First

**Speak naturally. Get instant text.**  
Works with **Hinglish + 10+ Indian languages**.

---

## 🚀 What It Does

- 🎤 **Voice → Text instantly**
- 🌐 **Auto-detects language** (Hinglish, Hindi, Tamil, etc.)
- 📱 **Mobile-first UI** (feels like a native app)
- 📝 **5-min continuous recording**
- 📂 **Local history (private, on-device)**

No setup friction. Tap **Speak → Done**.

---

## ✨ Why It’s Different

Most tools break on real-world Indian speech. This doesn’t.

- Handles **code-mixing (Hinglish)** cleanly  
- Designed for **actual usage**, not demos  
- Fast, minimal, no clutter
